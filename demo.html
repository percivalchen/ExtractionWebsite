Demo

Welcome to the ExtrAction user interface. This is a version of the interface that professionals will use when working with the ExtrAction extension. To demo, paste the text of your choice into the text box and enjoy the action items that are detected!


Click on the "?" to see a detailed guide to the extension.


* Note, the "Extract from Page" feature is not accessible in this demo because it requires you to have the gmail open and a functional chrome extension operating. To download our chrome extension, check out our technical page in the navigation bar.


If you would like to try the extension in your browser or if the demo isn't working but you would like to see our extension in action, please follow the steps listed here:

  1. Clone our repo here.
  2. In the address bar, navigate to chrome://extensions
  3. Toggle "Developer mode."
  4. Click "Load unpacked".
  5. Navigate to the cloned repo.
  6. Pin the extension to the browser tab.
  7. Test our extension!
